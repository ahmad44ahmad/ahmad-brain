أهلاً بك يا أحمد، سعيد بمساعدتك في تحليل مجلد **"2025 مؤشرات الأداء البداية"**.

  

يحتوي هذا المجلد على 21 ملفًا، تتنوع بين المستندات وجداول البيانات وعروض PDF، وجميعها تدور حول **حقيبة مؤشرات الأداء (KPIs) لمراكز التأهيل الشامل للعام 2025م**. تهدف هذه الحقيبة إلى توحيد منهجية قياس أداء المراكز على مستوى المملكة، وضمان تحقيق الأهداف الاستراتيجية للوزارة في مجالات الخدمات الاجتماعية والصحية والتشغيلية.

  

تم إصدار تعميم رسمي (الرقم: 28597، التاريخ: 6/7/1446هـ) من الوزارة يطالب إدارات المراكز بتحقيق مستهدفات هذه المؤشرات والوثائق الداعمة لها، مما يؤكد أهمية هذه الحقيبة في التقييم الرسمي للأداء (راجع: تعميم حقيبة مؤشرات الاداء لمراكز التاهيل الشامل للعام 2025م.pdf).

  

فيما يلي ملخص شامل لأبرز محتويات الحقيبة والمؤشرات التفصيلية:1. الأقسام المغطاة ونظام التقييم

  

تُقسم مؤشرات الأداء الرئيسية (KPIs) في الحقيبة إلى عدة أقسام رئيسية، ولكل قسم وزن نسبي محدد في التقييم الإجمالي للمركز، مع تحديد نطاقات تقييم الأداء (ممتاز، جيد جداً، جيد، ضعيف):

|  |  |  |
| :-: | :-: | :-: |
| القسم الرئيسي | الوزن النسبي (التقديري) | نطاقات تقييم الأداء (مثال) |
| القسم الطبي | 30% | ممتاز: 100-80% |
| قسم التمريض | 25% | جيد جداً: 79-60% |
| الصيدلية | 12% | ضعيف: \\\< 60% |
| الاجتماعي | \*\*(غير محدد الوزن)\*\* | ضعيف: \\\< 40% |

2\. مؤشرات الأداء التفصيلية (KPIs)أ. القسم الطبي (30% من الوزن الإجمالي)

  

تُركز مؤشرات القسم الطبي على سلامة المستفيدين والحد من المخاطر والأمراض المعدية:

|  |  |  |  |  |
| :-: | :-: | :-: | :-: | :-: |
| المؤشر | القطبية | المستهدف (مثال) | وزن المؤشر | المعادلة الرئيسية (للاحتساب) |
| عدد حالات التقييد خلال الشهر الحالي | \\+ | 100% | 4% | (20 - عدد حالات التقييد) / 20 \\\* 100 |
| عدد حالات الإصابات الجسيمة خلال الشهر الحالي | \\- | 0% | 10% | 100 - ((إجمالي المستفيدين - الإصابات الجسيمة) / إجمالي المستفيدين \\\* 100) |
| عدد حالات الأمراض المعدية الواجبة للإبلاغ | \\+ | 100% | 4% | (5 - عدد حالات الأمراض الوبائية المعدية الجديدة) / 5 \\\* 100 |
| العدد الفعلي لحالات الإصابات الناتجة عن السقوط | \\+ | 100% | 4% | (5 - عدد حالات الإصابات الناتجة عن سقوط) / 5 \\\* 100 |
| نسبة المستفيدين المحالين وتم قبولهم في نظام الإحالة | \\+ | 100% | 2% | عدد المقبولين / عدد المحالين \\\* 100 |

ب. قسم التمريض (25% من الوزن الإجمالي)

  

تُركز مؤشرات التمريض على جودة الرعاية السريرية ومتابعة الحالات المزمنة والطوارئ:

|  |  |  |  |  |
| :-: | :-: | :-: | :-: | :-: |
| المؤشر | القطبية | المستهدف (مثال) | وزن المؤشر | التجميع (الفترة الزمنية) |
| عدد نوبات الصرع التي تم مباشرتها خلال الشهر الحالي | \\- | أقل من 10% | 3% | مرحلي |
| عدد قرح الفراش لطريحي الفراش خلال الشهر الحالي | \\+ | 100% | 4% | مرحلي |
| عدد الحالات الطارئة التي تطلبت تدخل الهلال الأحمر | \\- | أقل من 10% | 5% | مرحلي |
| نسبة المستفيدين الذين تم عمل تحاليل مخبرية دورية لهم | \\+ | 100% | 2% | تراكمي - نصف سنوي |
| نسبة المستفيدين الذين تلقوا خدمات العيادات المتخصصة | \\+ | 100% | 2% | تراكمي - ربع سنوي |

ج. مؤشرات القسم الاجتماعي (تنمية المهارات والأنشطة والتواصل الأسري)

  

تُعد مؤشرات القسم الاجتماعي مؤشرات مركبة تشمل جوانب التأهيل والتمكين والعلاقات الأسرية، وتُجمع بياناتها في جداول ربع سنوية (راجع: نماذج مؤشرات الاداء للعام 2024 (القسم الاجتماعي).xlsx):

|  |  |  |  |
| :-: | :-: | :-: | :-: |
| المؤشر المركب | المؤشرات الفرعية الرئيسية | الوصف (المعادلة) | الوثائق الداعمة |
| \*\*تنمية المهارات\*\* | % تنفيذ الخطط التدريبية، % تدريب المستفيدين، % مشاركة الأسر. | يقيس تنمية المهارات الاستقلالية (كالأكل، استخدام الحمام) والاجتماعية (السلوكيات المقبولة، تعديل السلوك). (يتم وزن كل مؤشر فرعي، مثلاً 40% لتدريب المستفيدين). | استمارة التدريب |
| \*\*البرامج والأنشطة\*\* | % الأنشطة الداخلية، % الأنشطة الخارجية، % مشاركة الأسر، % مشاركة الجهات الخارجية. | يقيس تفعيل النشاطات الاجتماعية (المستهدف 192 نشاطاً سنوياً، بمعدل 12 داخلي و 4 خارجي شهرياً). (يتم وزن كل مؤشر فرعي بنسب مختلفة، مثل 30% للداخلية و 30% للخارجية، و 20% لكل من الأسر والجهات الخارجية). | نموذج رقم (1-3) |
| \*\*التواصل الأسري\*\* | % التواصل عبر الزيارات الخارجية، الداخلية، الاتصال المرئي، الاتصال الهاتفي. | يقيس نسبة التواصل الأسري للمستفيدين. (مثال: في يناير 2025 لمركز التأهيل الشامل بالباحة، العدد الكلي للمستفيدين 254، وتم التواصل مع 243 مستفيدًا، وتم التركيز على الاتصال المرئي بنسبة 35% من الوزن، يليه الزيارات الداخلية بنسبة 30%). \*\*(ملاحظة: مؤشر يناير 2025 بالباحة يوضح أن 143 مستفيداً خارج المنطقة.)\*\* | سجل الزيارات |

3\. الوثائق الداعمة ونماذج العمل

  

تتضمن الحقيبة نماذج موحدة ضرورية لتوثيق الأداء:

|  |  |  |
| :-: | :-: | :-: |
| النموذج / الوثيقة | الهدف الرئيسي | القسم المعني |
| التقرير الشهري لبيانات الأقسام الطبية والتمريضية (3) | نموذج شهري لجمع البيانات الأساسية لحالات التقييد، الإصابات، الأمراض المعدية، نوبات الصرع، قرح الفراش، والإحالات الطارئة. | الطبي والتمريض |
| نموذج تحويل للالتحاق بالبرامج التدريبية (رقم 4) | يوثق عملية تحويل المستفيدين للبرامج التدريبية، متضمناً التشخيص الطبي والنفسي والأهداف المرجوة من التحويل. | الاجتماعي |
| نموذج الخطة التربوية الفردية (رقم 5) | يُستخدم لتحديد الأهداف العامة والخاصة، وتحديد تاريخ البدء والانتهاء للخطة التدريبية لكل مستفيد. | الاجتماعي |
| استمارة الزيارات الميدانية (مركز التأهيل الشامل) | توثيق لبيانات المركز (مثل عدد النزلاء 254: 133 ذكور، 121 إناث)، والهيكل التنظيمي للموظفين الفعليين (مثال: أحمد عبد الله الشهري - رئيس قسم الجودة). | الإدارة والجودة |

هل ترغب في مقارنة أداء مركز التأهيل الشامل بالباحة في مؤشر معين (مثل التواصل الأسري لشهر يناير 2025)، أو التعمق في أي من الأقسام؟
