# Al-Baha Center Q3 2025 Facilities Quality Dashboard

## What this is

A live quarterly dashboard tracking **42 facilities-quality indicators** for "الإدارة العامة للمرافق" at Al-Baha center, with Q1/Q2/Q3 2025 status + values + notes captured. Mid-year revision pass — the kind of artifact [[mhrsd-2025-kpi-portfolio]] hypothesized exists ("a Q2/mid-year revised set is plausible") but couldn't surface.

The dashboard is structured as **two parallel tables**:

| Type | Count | Q3 status |
|---:|---:|---|
| Qualitative (مطبق/غير مطبق + notes) | **24** | 24/24 applied (100%) |
| Quantitative (numeric value + completion %) | **18** | 18/18 above 80% (avg 85%, all marked "قيمة تقديرية" / estimated) |
| **Total** | **42** | **100% application rate** |

## Quarter-over-quarter improvement signal

The dashboard captures **real progress** between quarters on 6+ indicators:

| Indicator | Q1 status | Q2 status | Q3 status |
|---|---|---|---|
| تمييز المسار الخارجي بأشكال وألوان | غير مطبق | غير مطبق | **مطبق** |
| صناديق البريد + أسماء السكان | غير مطبق | غير مطبق | **مطبق** |
| جرس مرئي (انترفون) | مطبق (قفل برقم بدون كاميرا) | طلب على نظام موارد | **مطبق + نُفِّذ** |
| إضاءة ذكية + تحكم صوتي | غير مطبق | غير مطبق | مطبق (مجدول للربع الرابع) |
| كاميرات مراقبة شاملة + 90 يوم حفظ | مطبق جزئيا | مطبق جزئيا | **مطبق + متابعة** |

**The Q3 dashboard documents real Q1→Q3 improvements**, not just steady-state operation. This is operational evidence that quality discipline + structured improvement plans (per the [[mhrsd-2025-albaha-branch-quality-plan]] mechanism) actually produces measurable change.

The "Health + Safety / care staff" category includes PT-relevant indicators ("توفر اخصائي العلاج الطبيعي خاصة بترخيص وزارة الصحة" — availability of MoH-licensed PT specialists). **Ahmad's MoH licensure satisfies this indicator.**

## Disability-accessibility lens

The dashboard is heavily weighted toward **physical accessibility** — entrance ramps, door widths, parking, signage, elevator (1 hydraulic, 3-4 stops), Braille + tactile maps for vision impairment, anti-slip flooring, controlled lighting at room entrances.

**Two indicators expose specific Q3-level gaps in vision-impairment support:**

- *لغة برايل + خرائط لمسية لكف البصر* — Q3 listed as مطبق but notes say "سيتم طلب الاحتياج علي نظام موارد و مجدولة للربع الرابع" (still in procurement, scheduled for Q4). Status-vs-notes inconsistency.
- *إضاءة ذكية + تحكم صوتي + إستشعار حركة* — same pattern.

These are the kind of "applied but not really" indicators where dashboard hygiene matters. **The center is over-marking مطبق to hit 100% application target while the notes column tells the real story.** Worth flagging as a data-integrity pattern in any [[basira]] argument that real outcomes need real measurement, not just status checkboxes.

## Quantitative scoring caveat

All 18 quantitative indicators show:
- Q1 = Q2 = 0 (not activated)
- Q3 = 85, marked **قيمة تقديرية** (estimated value)

**Q3 quantitatives are placeholder estimates, not measured values.** This is honest signaling — the center hasn't yet operationalized data collection for the quantitative side. The qualitative side is doing real work; the quantitative side is performative.

For [[basira]] credibility: flag this. A digital-transformation pitch that promises *real* quantitative measurement (vs. quarterly "تقديرية" placeholders) hits a clear pain point.
