# Basira — Sovereign Decks

## Why three, not one

Ahmad built a three-tier audience ladder, not a single generic deck. Each tier has its own visual vocabulary, rhetorical structure, and register matching the audience. Showing the wrong tier to the wrong audience is worse than showing nothing — Tier 1 to a minister feels operational; Tier 2 to a cybersecurity director feels too lofty; Tier 3 outside its audience is category-confused.

## Tier 1 — Governance (for deputies and wakalat)

- **File:** `1-governance-wikala.html`
- **Audience pill:** «مُوجَّه لسعادة الوكلاء وأصحاب المصلحة»
- **Aesthetic:** Corporate / institutional. IBM Plex Sans Arabic + Space Mono. Teal `#006F72` and gold `#B88A2E`. Grid-based KPI tiles, compliance-matrix layout.
- **Spine:**
  1. Four-KPI strip — **SROI 1:4.2 · ISO 9001 100% · Audit Trail 100% · Scale 1→36**.
  2. Four pillars (Dignity + Connection · Intelligence + Prediction · Safety + Operations · Governance + Risk + Compliance) with code refs C1-01..C4-03.
  3. Vision 2030 alignment bars — مجتمع حيوي 92% · اقتصاد مزدهر 78% · وطن طموح 96% · جودة الحياة 88%.
  4. Compliance matrix (6 tiles) — ISO 9001:2015, NCR/CAPA, Audit Trail, Emergency Plan, حِصن, Red Crescent 5-Stars.
  5. Reference code `BSR/GOV/5.0/2026-04` + classification "عرض تنفيذي مُوجّه".
- **Audiences:** الإدارة العامة للجودة ([[ismail-alghamdi]]), وكالة التحول الرقمي, إدارة الامتثال, وكالة تجربة المستفيد, cybersecurity technical review, وكالة التأهيل والتوجيه ([[nasser-alqahtani]], CC).

## Tier 2 — Sovereign (for minister and minister's assistants)

- **File:** `2-strategic-minister.html`
- **Audience pill:** «عرض سيادي لمعالي الوزير ومساعدي أصحاب المعالي»
- **Aesthetic:** Sovereign / royal. Noto Kufi Arabic + Tajawal. Saudi-flag green `#006C35`, gold `#C8A95C`, cream `#FAF6ED`. Pyramidal tiered layout with a radial gold glow.
- **Spine (pyramid):**
  1. **Tier 01 — The anchor:** Vision 2030 statement ("مواطن ممكن، ومجتمع كريم").
  2. **Tier 02 — The killer number:** SROI **1:4.2** with contextual paragraph. The single most load-bearing element of the deck.
  3. **Tier 03 — The four pillars** (visual recap).
  4. **Tier 04 — Scale story:** 1 → 36, Al-Baha pilot to kingdom-wide.
  5. **Decision callout:** «هل نُطلق بصيرة 5.0 كنموذج وطني... الآن، أم ننتظر دورة تقييم إضافية؟»
  6. **Closing equation:** بصيرة = كرامة + كفاءة + حوكمة.
  7. Footer — Al-Baha center + MHRSD.
- **Strategic note:** Tier 2 ends on a forced binary ("launch now vs wait"). It is a **decision deck**, not a status deck — deliberate, per the helper-over-display principle.
- **Audiences:** معالي الوزير (أحمد الراجحي), نائب الوزير للخدمات المشتركة, نائب الوزير لقطاع التنمية, any وكيل وزارة.

## Tier 3 — Dignity / Wajaha (for social-fabric audiences)

- **File:** `3-dignity-wajaha.html`
- **Audience pill:** «موجَّهٌ لأصحاب الوجاهة وشيوخ القبائل وأعضاء الأسرة الحاكمة»
- **Aesthetic:** Heritage / Islamic. Amiri (serif, italic) + Reem Kufi. Sand, gold, najdi-red `#7A1E22`, deep green `#1F4E3D`, cream. Ornamental Qur'anic framing, arabesque borders.
- **Spine:**
  1. Qur'anic frame — ﴿وَلَقَدْ كَرَّمْنَا بَنِي آدَمَ﴾ — سُورَةُ الإِسْرَاء، الآية ٧٠.
  2. Three-era triptych — الأمس (رعاية صامتة) → اليوم (كرامة مُنظَّمة) → الغد (مواطن مُمكَّن).
  3. Dignity Mandala (SVG): concentric rings — الإنسان → الأسرة → المركز → المجتمع → الوطن.
  4. Three-virtues trio — إحسان (Dignity Profile) · صِلَة (Family Connect 2.0) · أمانة (Audit Trail), each paired with a hadith or verse.
  5. Royal-patronage closing — «بُنيَت بصيرة بأيدي أبناءِ البلد، ولخدمة أهلِ البلد، بتوفيقٍ من قيادةٍ رشيدة».
  6. Colophon — بصيرة 5.0 + MHRSD + Al-Baha.
- **Critical:** Tier 3 is NOT for ministry review. Do not send it to cybersecurity or compliance offices.
- **Audiences:** أصحاب الوجاهة الاجتماعية, شيوخ القبائل, أعضاء الأسرة الحاكمة في الباحة, any royal-level patron meeting.

## How to use the decks

**In correspondence.** Attach the matching tier's PDF export to the body of the letter: to Al-Ghamdi or any deputy, Tier 1 + Tier 2; to the minister's office, Tier 2 only; to regional royalty or social patrons, Tier 3.

**In meetings.** Open with the right tier. Keep the footer reference codes (`BSR/GOV/5.0/2026-04`, etc.) — they signal document control and archival readiness.

**For video production.** Each deck is already a mini-narrative. A four-minute voiceover reading of Tier 2 slides produces a sovereign-grade video; Tier 1 sections map to a technical walkthrough video.

## Conversion

- **HTML → PDF:** `scripts/html-to-pdf.mjs` in the Basira repo (Playwright-based, tested for the 19-page operational PDF).
- **HTML → MP4:** screen-record via OBS or PowerPoint Record while scrolling. Verify the output before shipping.
