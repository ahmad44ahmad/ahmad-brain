# Basira Evidence Stack

## Why this page exists

The Basira pitch arc rests on four claims:

1. **Governance maturity.** "We already run a regulated, audited operation."
2. **Measurement infrastructure.** "We already measure performance — and it's fragmented in ways Basira solves."
3. **Empowerment alignment.** "Our work is dignity-and-community first, not custodial."
4. **Escalation capability.** "We already act decisively when standards are at risk."

Across the vault, 14 notes carry the `basira-evidence-stack` tag. Each anchors one or more of these claims with primary-evidence Drive sources. This page maps them so any pitch construction starts with the strongest evidence cluster, not a blank page.

2. **The MHRSD KPI surface already encodes empowerment metrics** — beneficiary satisfaction, vocational empowerment retention, deinstitutionalization. Basira amplifies what exists; doesn't invent.
3. **Quality oversight is a two-layer structure**: branch (Khalid Al-Zahrani + Mariya Al-Mohammedi) and center (Ahmad). Both layers in alignment provide political cover for any Basira pitch.
4. **Real Q1→Q3 improvement is documented** ([[mhrsd-2025-albaha-q3-facilities-dashboard]]) — quality discipline produces measurable change. Basira's claim that systematic measurement helps is *already proven* by Ahmad's center.
5. **Five named operational gaps** (no rollup, "تقديرية" placeholders, paper logs, status-vs-notes inconsistency, paper escalation routing) tie Basira features to specific real pain points — not abstractions.
