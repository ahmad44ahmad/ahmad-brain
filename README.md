## Plugins required

Install from Obsidian → Settings → Community plugins:

1. **Templater** — powerful templates (daily notes, project skeletons)
2. **Dataview** — query vault by frontmatter
3. **Tasks** — checkbox queries anywhere
4. **Obsidian Git** — auto-commit every 30 min (critical for version safety)
5. **Omnisearch** — better full-text search

Configure Templater → Template folder = `90-Meta/templates`.
